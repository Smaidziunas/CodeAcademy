Tag'ų sąrašas

  • h1-h6 - Antraštė
  • p - Paragrafas
  • img - Paveiksliukas
  • a - Nuoroda
  • div - Div'as
  • ul - Nerikiuotas
  • ol - Rikiuotas
  • li - Sąrašo elementas
  • dl - Aprašomasis sąrašas
  • dt - Aprašomojo sąrašo terminas
  • dd - Aprašomojo sąrašo termino paaiškinimas
  • hr - horizantali linija
  • br - Eilutės lūžis

Atributų sąrašas

  • src - Šaltinis į nuorodą, failus
  • href - Nuoroda
  • width - Plotis
  • height - Aukštis

Klaviatūros trumpinių sąrašas

  • SHIFT ALT F - Viso failo Tabuliavimas
  • ctrl + arrows - Vaikščiojimas per žodžius
  • ctrl + SHIFT + arrows - Viso žodžio pažymėjimas
  • HOME - nukelia į eilutės pradžią
  • END - nukelia į eilutės pabaigą
  • ALT + TAB - Pereina į kitą aplikaciją
  • Window V - Atidaro ClipBoard
  • Ctrl C - Copy
  • Ctrl V - Įklijuoti nukopijuotą tekstą
