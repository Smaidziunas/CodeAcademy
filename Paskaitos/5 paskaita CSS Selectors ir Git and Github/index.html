CSS Selectors

Visus CSS Selector'ius galima rasti čia

  • element - Pagal Tag'o vardą.
  • class - Pagal tag'ui suteiktą klasę
  • id - Pagal tag'ui suteiktą id
  • .class.class - tag'ui suteiktos abi (arba daugiau) klasės
  • .class3 .class4 - tag'ui suteikia stilius jeigu jis turi class4 klasę ir yra class3 palikuonis
  • * - parenka stilius viskam
