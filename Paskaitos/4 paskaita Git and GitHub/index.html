Naudojant bet kurią apart Statinės pozicijos, gauname priegą prie 5 naujų stilių: top, left, right, bottom, z-index.
Šitie stiliai nurodo per kokį atstumą elementasturi būti atitrauktas.

Terminalo GitBash komandos

  • cd - change directory - pakeisti direktoriją - Vaikščiojimas tarp aplnakalų
  • .. - grižta atgal
  • ls - list files and directories
  • CTRL C - atšaukia vykdomą veiksmą
  • TAB - Užbaigia frazę arba pateikia pasiūlymus
  • clear - nukelia visą ekrane rodomą informaciją į viršų
  • rm -fr "FILE NAME" - Panaikina Master Repozitorija
  • code . - atidaro visus folders ir failus toje aplinkoje ir nustatyta code editor sistemą

Git Config

  • init - initialize - inicializuoti
  • status - parodo informaciją apie failus esamoje repozitorijoje
    Tuščius aplankus ignoruoja
  • add - stage'ina/prideda/paruošia failus commit'inimui, arba add ir (.), kad įdėtų visus failus
  • commit -m "Comm'ito žinutė"
  • push - įstumia lokalius, jau sukomitintus pakitimus, perduoda GitHub
  • pull - įkelia Github informaciją į lokalią aplinką
