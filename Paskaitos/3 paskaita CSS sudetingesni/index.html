Puslapio Struktūra

  • header - puslapio viršus, kuriame nurodoma informacija apie puslapį / MENU
  • nav - skirtas puslapio MENU
  • section - didžiulė puslapio dalis
  • div - maža puslapio dalis
  • main - pagrindinė puslapio dalis
  • hero - dažnai //Main// apie ką ura puislapis
  • article - dažniausiai //Main// viduje esanti sekciija
  • aside - dažnai //Main// viduje, šoninė puslapio sekcija
  • footer - puslapio apačia, kurioje yra įvairi informacija kaip: TCs; Privacy Policy; Copyrights; links; partners...
  • li:nth-child(numeracija) - numeracija nurodytas kas kažkelintas vaikinis elementas

Menu Info

Kuriant Menu - naudojamas NAV tag'as, kurio viduje yra rikiuotas/nerikiuotas sąršas (ol/ul). To sąrašo eslemtu (li) viduje yra nuorodas tag'as (a), kuris nurodo į kažkurią puslapio dalį arba kitą puslapį.

Nuorodos tag'o (a) atidaromojoje dalyje esantis atributas href nurodo į kur vartojas turi būti nukeltas paspaudus ant nuorodos.

Jeigu href atributas nurodo į kažkokį ID (href="aplankas"), tuomet vartotojas nukeliamas į nurodytą html failą Pvz..: Kitas Puslapis

Inline, Block ir Inline-block

Inline

Inline (Linijiniia) elmentai yra tokie elementai, kurie užima tik tiek vietos ekrane, kiek jiems yra reikalinga. Minusas yra tas, kad negalima keisti šių elemntų dydžių. Pliusas tas, kad jie įtalpinami tarp kitų informacijos vienetų.
Tag'ų pavyzdžiai: a, span, img, daugiau pavyždžių: W3 schools.

Block

Block elementai yra tokie elementai, kurie užima visą leistiną plotį ir neisileidžia kitų elementų į tą pačią eilutę. Pliusas, kad galima keisti jų dydį.
Tag'ų pavyzdžiai: h1-h6; li; p; div; section; main; article; nav; ol; ul; blockquote; aside; canvas; table etc..: W3 shools.

Inline-Block

Inline-Block elementai yra tokie elementai, kurie pasisavino geriausias Blokinių ir Linijinių elementų savybes. Jie užima tik tiek vietos, kiek jiems reikia, bet jų dydį vis tiek galima keisti.

Model Box

Tarp elementų dėl "White Spacing" atsiranda tarpas, kurį galima panaikinti, ištrynus tarpus tarp elementų kodų, kodo editor aplinkoje.

Margin - Atstumas nuo elemento rėmelių iki aplinkiniu elementų rėmelių (skirtingų elementų margin'ai persidengia).

Border - Rėmelis, kuris yra tarp Margin ir Padding.

Padding - Atstumas nuo elemento content iki elemento rėmelio. Du padding'ai skirtingų elementų niekada nepersidengia, margin'nų žemyn aukštyn žemyn persidengia o į šonus margin'ai nepersidengia.

Content - Eelemento tūrinys (blokinio elemento tūrinbžys užims visą leidžiamą eilutę arba nurodytą dydį).

Galime aprašytt Margin, Border, ir Padding nustatymus 4 skirtingais būdais.
Vienas matmuo visom 4 kraštinėm pvz..: (margin: 50px)
Rašant dvejus matmenis - pirmasis priskiriamas - viršuj ir apačiai, antras - šonams.
Rašant tris matmenis, pirmasis - viršuj, antras - šonams, trečiasis - apačiai pvz..:(padding: 30px 20px 10px).
Rašant keturis matmenis, jie priskiriami pagal laikrodžio rodyklę, eilės tvarka, pradedant nuo viršution matmens.
